Docker

How to run in docker

Run this command in your terminal：

bash <(curl http://download.mlsql.tech/scripts/run-all.sh)

Notices:

  1. Your system should be Linux/Mac OS, and have docker installed.
  2. 9002,9003,8080,3306 should be not taken by other programs.
  3. If there are some MySQL errors, ignore them, the script is checking when the mysql is ready.

Using docker ps to check the result:

All docker containers are up.

Notice

We disable the table auth by default since when this is opened which will make you not easy to use. You can enable it when you start MLSQL console Docker container with adding "-e ENABLE_AUTH_CENTER=true".

#!/usr/bin/env bash

java -cp .:${MLSQL_CONSOLE_JAR} tech.mlsql.MLSQLConsole \
-mlsql_cluster_url ${MLSQL_CLUSTER_URL} \
-my_url ${MY_URL} \
-user_home ${USER_HOME} \
-enable_auth_center ${ENABLE_AUTH_CENTER:-false} \
-config ${MLSQL_CONSOLE_CONFIG_FILE}
